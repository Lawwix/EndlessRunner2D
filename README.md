# EndlessRunner 

**Unity endless runner project**

## Описание проекта
EndlessRunner — это учебный проект по созданию игры жанра *endless runner* на Unity.  
Игрок управляет персонажем, который бежит вперёд, избегает препятствий и собирает очки.  
Цель игры — продержаться как можно дольше, не столкнувшись с препятствиями.

## Особенности
- Постепенное ускорение персонажа с течением времени.
- Подсчёт очков.
- Простое меню и экран конца игры.
- Оптимизация под ПК и WebGL.

## Технические детали
- **Игровой движок:** Unity 2022.3 LTS*
- **Язык программирования:** C#
- **Среда разработки:** Unity Editor + Visual Studio / Rider
- **Целевая платформа:** Windows / WebGL

## Структура проекта
Assets/
├── Scripts/        # Скрипты на C#
├── Prefabs/        # Префабы (игровые объекты)
├── Materials/      # Материалы и текстуры
├── Scenes/         # Сцены Unity
├── UI/             # Элементы интерфейса
└── Audio/          # Звуки и музыка

## Как запустить проект
1. Склонируйте репозиторий или скачайте ZIP:
   https://github.com/Lawwix/EndlessRunner2D.git
2. Откройте папку проекта через **Unity Hub**.
3. Запустите сцену `MainScene.unity`.
4. Нажмите **Play** для запуска игры в редакторе.
